# Design Trend - CSS3 Neumorphism Buttons

Um projeto simples de **botões com efeito neumorphism** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar **botões modernos com estilo neumorphism**, utilizando sombras suaves para um visual elegante e minimalista.

## 🛠️ Tecnologias

- HTML
- CSS (box-shadow, neumorphism)

## 🎯 Funcionalidades

- Botões estilizados
- Efeito neumorphism
- Design moderno e minimalista

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
